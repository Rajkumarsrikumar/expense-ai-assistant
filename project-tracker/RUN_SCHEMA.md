# Fix: Create the `projects` table

**Important:** Paste the SQL code below, NOT a URL. The URL is only for opening the Supabase page.

## Steps

1. **Open Supabase SQL Editor** (in your browser, go to supabase.com → your project → SQL Editor → New query)

2. **Copy the SQL** – Either:
   - Open **http://localhost:5173/copy-schema.html** and click "Copy to clipboard", OR
   - Open `supabase/schema.sql` or `public/schema.sql` in this project and copy all contents

3. **Paste into the SQL Editor** – Paste the SQL code (starts with `CREATE EXTENSION`), NOT any URL

4. **Click Run** (or Ctrl+Enter)

5. **Refresh your app**

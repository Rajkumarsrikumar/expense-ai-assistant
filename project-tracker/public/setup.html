Supabase Setup – Project Tracker

1. Create a Supabase project

  1. Go to supabase.com/dashboard
  2. Click New Project
  3. Enter name, database password, and region
  4. Click Create new project and wait 1–2 minutes

2. Run the database schema

  1. In your project, open SQL Editor (left sidebar)
  2. Click New query
  3. Open supabase/schema.sql in this project folder
  4. Copy all contents and paste into the SQL Editor
  5. Click Run (or Ctrl+Enter)

3. Get your API credentials

  1. Go to Settings → API
  2. Copy Project URL
  3. Copy anon public key

4. Configure the app

Edit the .env file in the project root:

VITE_SUPABASE_URL=https://your-project-id.supabase.co
VITE_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

Replace with your actual values. Then restart the dev server.

5. Create test user (optional)

  1. Go to Authentication → Users
  2. Click Add user → Create new user
  3. Email: testuser@example.com, Password: Test123456!
  4. Enable Auto Confirm User

6. Disable email confirmation (optional)

For easier testing: Authentication → Providers → Email → turn off Confirm email.
